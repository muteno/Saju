# 유튜브 채널 → 전체 영상 링크 추출 (2가지 방법)

특정 유튜버 채널의 **모든** 영상 링크를 뽑는 법. (내가 스크롤한 부분만이 아니라 채널 전체 기준)

---

## 방법 A ⭐ 추천 — yt-dlp 한 줄 (설치돼 있음, 스크롤 불필요·누락 없음)

터미널(PowerShell)에서 채널 주소만 바꿔 실행:

```powershell
# 채널의 '동영상' 탭 전체 → 링크를 txt로
yt-dlp --flat-playlist --print "https://www.youtube.com/watch?v=%(id)s" "https://www.youtube.com/@채널핸들/videos" > links.txt
```

- `@채널핸들` 자리에 그 채널 핸들(예: `@chocosd`)이나 채널 URL을 넣으면 됨.
- **동영상 + 쇼츠 + 라이브 전체**를 원하면 탭을 나눠서:
  ```powershell
  yt-dlp --flat-playlist --print "https://www.youtube.com/watch?v=%(id)s" "https://www.youtube.com/@채널핸들/videos"  >  links.txt
  yt-dlp --flat-playlist --print "https://www.youtube.com/watch?v=%(id)s" "https://www.youtube.com/@채널핸들/shorts" >> links.txt
  yt-dlp --flat-playlist --print "https://www.youtube.com/watch?v=%(id)s" "https://www.youtube.com/@채널핸들/streams" >> links.txt
  ```
- 재생목록도 동일: URL 자리에 `.../playlist?list=PL....` 넣으면 그 목록 전체.
- 채널의 **업로드 전체**를 한 번에: 채널 메인 URL 또는 `@핸들` 만 줘도 yt-dlp가 펼침.

이 방법이 가장 빠르고(수 초) 스크롤·DOM 문제 없이 **누락 0** 입니다.

---

## 방법 B — 브라우저 개발자 콘솔 (설치 없이 브라우저에서)

채널의 **동영상 탭**(`https://www.youtube.com/@채널핸들/videos`)을 연 뒤, F12 → Console 에 아래를 붙여넣고 Enter.
스크롤을 **자동으로** 끝까지 내리며 수집하므로, 내가 안 내린 영상까지 전부 잡힙니다. (끝나면 클립보드에 복사됨)

```javascript
(async () => {
  const seen = new Set();
  const grab = () => document.querySelectorAll('a[href*="/watch?v="]').forEach(a => {
    const m = a.href.match(/[?&]v=([\w-]{11})/);
    if (m) seen.add(m[1]);               // DOM이 재활용되기 전에 매 스크롤마다 수집
  });
  let last = -1, stable = 0;
  while (stable < 4) {                    // 4번 연속 증가 없으면 종료
    grab();
    scrollTo(0, document.documentElement.scrollHeight);
    await new Promise(r => setTimeout(r, 1200));
    if (seen.size === last) stable++; else { stable = 0; last = seen.size; }
  }
  grab();
  const urls = [...seen].map(id => 'https://www.youtube.com/watch?v=' + id);
  copy(urls.join('\n'));                  // 콘솔 전용 copy() → 클립보드
  console.log(urls.join('\n'));
  console.log('✅ 총 ' + urls.length + '개 링크 복사됨 (클립보드)');
})();
```

- **핵심**: 유튜브는 화면 밖 영상의 DOM을 지워버리므로(가상 스크롤), 끝까지 내린 뒤 한 번에 긁으면 앞부분이 사라집니다. 위 코드는 **스크롤할 때마다 수집**해서 누락을 막습니다.
- 쇼츠/라이브는 각 탭(`/shorts`, `/streams`)에서 같은 코드를 다시 실행.
- 복사된 링크를 `links.txt`로 저장하면 방법 A/스크립트에 그대로 투입 가능.

> 방법 A(yt-dlp)가 더 빠르고 확실합니다. 콘솔 방법은 yt-dlp를 못 쓰는 환경용 백업.
