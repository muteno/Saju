# 지식 베이스 (L2) — 결정론 조회 색인

**절대 원칙 구현체**: 자료를 찾는 주체는 LLM이 아니라 사주 데이터다.
L1 만세력 엔진이 낸 키(일주·십신·신살·합충…)로 이 색인을 **그대로 조회**한다. 검색·임베딩 없음.

## 파일

- `unit_index.json` — 요소 키 → 코퍼스 유닛 목록(`{key, doc, title}`). `kb-tools/map_units.py`가 생성.
  유닛 key는 `refine-tools/units.json`의 key(`문서명#순번`)와 동일 — 본문은 그 문서에서 추출.
- `aliases.json` — 엔진 방출 키 → 색인 키 별칭/폴백 체인. 근거 유닛이 없으면
  "소장 문헌에 상세 없음" 처리(추측 서술 금지).
- `unmapped_report.md` — 요소 문서인데 매핑 안 된 제목(수동 검토용).
