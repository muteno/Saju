# manse/db — 만세력 정적 DB (생성물)

여기 있는 JSON들은 **손으로 만들지 않았다.** `manse/js/knowledge`(지식모듈, 정본)에서
`manse/tools/build-db.mjs`가 기계로 뽑아낸 **파생 테이블**이다.

> 쉽게: 규칙(공식)은 js 모듈에 한 벌만 두고, 그걸 모든 조합으로 미리 계산해 표로 구운 것.
> 그래서 **여기 파일을 직접 고치면 안 되고**, 규칙을 고치고 싶으면 knowledge 모듈을 고친 뒤
> 아래 한 줄을 다시 실행하면 된다.

```bash
node manse/tools/build-db.mjs
```

## 쓰임새

- 이 JSON만 있으면 **자바스크립트가 아닌 환경**(파이썬·스프레드시트·서버 DB)에서도 같은 판정을 재현할 수 있다.
- 일진(날짜→간지)은 표가 아니라 공식이 정본: `(율리우스일수 + 49) mod 60` → `ganji60.json`의 idx로 조회.
- 규모가 커져 서버 DB가 필요해지면 이 JSON을 Cloudflare D1(SQLite)로 그대로 부어 넣는 경로를 권장
  (스키마 = 파일당 테이블 1개). 지금 단계에선 정적 JSON이 가장 단순하고 충분하다.

생성 때마다 앵커 7종(甲子·공망·십신·운성·신살·절기 행수·입춘 2000 날짜창)을 통과해야 파일이 써진다 —
실패하면 생성 자체가 중단된다.
