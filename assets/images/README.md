# assets/images

영화 「탈예울」 목업용 이미지. **전부 CC0(퍼블릭도메인)** — Openverse 로 검색해 받았습니다.
저작권 표기 의무는 없으나 출처를 남겨 둡니다(`CREDITS.json` 참고). 사용자가 직접 만든
키 아트가 준비되면 동일 파일명으로 교체하면 됩니다.

| 파일 | 용도 | 비고 |
|---|---|---|
| `001-hero-poster.webp` | 히어로 포스터(3:4) | 다크 오피스 |
| `002-cityscape-night.jpg` | 히어로/크레딧 배경 텍스처 | 도시 야경 |
| `003-exit-sign.webp` | 장식 액센트(EXIT) | 비상구 사인 |
| `004-empty-desk.jpg` | 예비 | 빈 책상 |

파일명 규칙: `NNN-<kebab-name>.<ext>` (코드의 `NNN-name.kind.js` 규칙과 동일한 넘버링).
시네마틱 톤 통일을 위해 표시 시 CSS 로 그레이드(어둡게 + 코발트 오버레이) 처리합니다.
