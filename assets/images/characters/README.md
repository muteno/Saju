# assets/images/characters

사주 **십이지(十二支)** 귀여운 마스코트 **12종**. 「탈예울」 콘텐츠용.

- **색 규칙**: 계절 대표색(**오방색**) — 봄靑·여름赤·가을白·겨울黑 / 中央黃=태강(안내자).
- **공통 시그니처**: 노란 나비넥타이 · 큰 반짝 눈 · 정면 앉은 포즈 · 두꺼운 진회색 외곽선.
- **생성/가공**: Gemini(구독형)에서 **계절 시트 4장(3마리씩)** 생성 → PIL/scipy 스크립트로 **자동 컷아웃**(바깥 배경 + 카드 흰색 제거 → 최대 연결덩어리만 유지 → 투명 정사각).
- **포맷**: 640×640 **투명 webp**(q90, 평균 ~50KB). 프롬프트 정본 = `docs/사주캐릭터_프롬프트팩.md`.
