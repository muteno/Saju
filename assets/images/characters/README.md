## gapja/ — 60갑자 색 변주 (60종)

- **생성(260712)**: 12지 컷아웃을 **천간 오행색(클로신 `--el-*` 기틀 토큰, 신규 색 0)**으로 휘도 보존 듀오톤 재착색 — 외곽선·눈 유지, 몸색만 변주. 음간(乙丁己辛癸) = 같은 색 +28% 연변주(양/음 구분).
- **파일명**: `NN-<gan>-<animal>.webp` (NN = 전통 60갑자 순서 01 갑자 … 60 계해 · 320px · 평균 18KB · 합 ~1MB). 예: `43-byeong-horse.webp` = 병오(丙午) — 올해.
- **용도**: 간지 색매칭(예 병오년 = 붉은 말) · 만세력 지지 아이콘 · 수집 요소. ⚠ 아직 UI 미배선 — 배선 시 필요한 것만 PRECACHE 등재(60장 전부는 과중).
- **품질 참고**: 프로그램 재착색이라 소품·볼터치도 함께 물듦(이모지 크기용 v1). 최상급 필요 시 `docs/사주캐릭터_프롬프트팩.md` §1-C 프롬프트로 Gemini 재생성.
