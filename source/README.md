# source — 작품 재료 모음

이 폴더는 **작품의 씨앗이 되는 원재료**를 모아두는 곳이다.
지금은 팀즈(Microsoft Teams) 대화를 자동으로 긁어와 여기에 쌓는다.

```
source/
├─ chats/      ← 모아진 대화가 여기에 쌓인다 (날짜별 마크다운 + 원본 JSON)
└─ _sync/      ← 대화를 자동으로 가져오는 도구 (설정·스크립트·설명서)
```

## 흐름 한눈에

1. 팀즈에서 참여자들과 대화한다.
2. 내 PC가 일정 시간마다 그 대화를 자동으로 가져와 `chats/`에 저장하고 깃에 올린다.
3. 우리는 `chats/`에 쌓인 걸 보면서, 마음에 드는 결을 골라 작품(시·장면 등)으로 다듬는다.

> **수집은 자동, 작품화는 사람 손으로.** 여기 쌓인 건 "날것"이다.
> 그대로 사이트에 올리는 게 아니라, 보고 골라서 다듬는 재료다.

## 처음 세팅하려면

`_sync/SETUP.md`를 그대로 따라 하면 된다. (PC를 다루는 단계가 있어서, 그 부분은 개발 담당이 맡는 걸 권한다.)

## 한 가지 약속

여기 쌓이는 건 **참여자들의 실제 대화**다. 모으기 전에 참여한 사람들에게
"우리 대화를 작품 재료로 모아둘게, 괜찮을까?" 하고 한 번 물어보고 시작하자.
