## 2. 핵심 반전 — 성향의 "출처"가 다르다

화법 §4·§6-2가 곧 "같은 사실을 청자 기질에 맞춰 포장". 결정적 차이:

> **Saju는 성향을 에니어그램 검사로 안 물어본다. 사주 자체에서 계산된다.**
> 명리가 이미 성향 이론이다 — 십신 분포(주체/수용/표현/실리/규범)·신강약·오행 편중·생왕고(역마·도화·화개). 팔자 넣으면 청자 기질축이 나온다.

→ 예타에서 가져올 건 **"성향 프로파일 → 말투 조절"이라는 기계**고, 성향의 소스는 예타(캐릭터 tune)가 아니라 **Saju 엔진**.

## 4. 성향적 활용 전체 그림

- **성향 소스** = 사주 엔진(십신·신강약·생왕고) → 청자 성향 프로파일.
- **성향 데이터(이미 Saju 보유)** = `자막흡수지도 §3`(일간 10종 성정) + `006 십신 키워드`(십신 그룹 성정).
- **조절 기계** = 예타 tune→band 방법론 이식 → 프로파일을 딜리버리 파라미터로.
- **가드** = 성향 라벨 겉에 안 드러냄(둘 다 백스테이지).

→ **같은 사주 사실을 그 사람 기질에 맞는 톤으로.** 예타가 사람 성향 맞춰 대화하듯, 도사가 내담자 기질 맞춰 통변. **예타는 기질을 데이터로 주는 게 아니라(그건 사주가 줌), 기질을 말투로 바꾸는 변환기·방법론을 준다.**

**정직한 한계**: 예타가 기계·방법론·가드는 통째로 주지만, **청자축별 실제 문안**(안정형 도사가 뭐라 말하나)은 사주 도메인으로 새로 써야 함. 단 원료(십신 성정·일간 성정)도 이미 Saju 안 → "맨땅" 아님, 배선만 남음.
