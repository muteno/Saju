재현 한계 각주 (정직 모드)

  • 테마 = 클로신 기틀 계승. 색·글래스·라운드·모션 토큰 = manse/app/app.css :root 사본(PR #21 브랜치 실측) — 오행색 --el-* 5종·코랄 --accent 포함, 신규 색 0. 십신 라벨·時日月年 배열·오행 색상원 구조 = 운영자 제공 레퍼런스(FACETELLER류) 참조.
  • 만세력 = 데모용 고정값. 생일을 바꿔도 팔자가 같음 — 실앱은 manse 계산 모듈 연결 자리. 데모 팔자는 대사와 정합(寅=역마 · 子·午=도화 · 金 0=금약).
  • 십신 = 실규칙 계산값(고정 일간 戊土 기준: 甲=편관·子=정재·丙=편인·寅=편관·戊=비견·午=정인·壬=편재·戌=비견). 지지는 본기 기준 단순화.
  • 일러스트 = CSS 플레이스홀더. 확정 웹툰 이미지(드래프트 §4 프롬프트)로 교체하는 자리. 표정 5종 슬롯 배선됨.
  • 대사 = 무-LLM 고정 스크립트. 실앱 = 템플릿 사다리 + LLM 승격(드래프트 §3). 선택지 분기 구조는 실설계와 동일.
  • 저장/공유 버튼 = 연출만. BGM·햅틱 없음(햅틱 = 시각 펄스 흉내).
  • 타이포 기준(운영자 확정 260712): 본문 = Pretendard · 자간 -10(=-0.01em) 전역 기본. 눈썹 라벨(광자간)·한자 인장(세리프)만 의도적 예외.
  • ?fast=1 = 검증용 고속 모드. 설정 축은 연출/레이아웃만(테마 축 없음 — 기틀 단일 계승이라).
