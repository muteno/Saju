## 3. /feed v2 로컬 적용 변형 (스킬 정본 준수)

- **대사 발화 단위 큐레이션 파일럿 3편**(C07·C08·C04 각 1, 전부 Whisper 신규) — 10분 컷 내 전원 완주. 산출 = 유닛 후보 **40개**(개념14·조건7·절차6·사례3·관점10), 축자 인용 전건 원문 대조(1편은 기계 substring 검증 PASS), **오전사 교정 64쌍**, 이견 2건(관점 병기 대상), 신규 키 후보 17(재물축 9 + 운이론축 8 — 기존 키 부재 grep 실증), 스킵률 ~23~25%(인사·홍보). ※시트 본문 = 답안지 파생물 → 레포 미커밋(세션 스크래치), 파도 1 루프가 동일 절차로 재생성·검증 후 정식 주입.
- **파일럿 실증 2건**: ①Whisper 환각 실구간 적발(동일 문장 반복 = O2 경고 실증 — 해당 구간 인용 채택 금지 처리) ②대본 내 표(조견표) 오전사 오염 발견 → 표준 이론값 대조 교정으로 복구(축자 인용 불가 구간은 명시 후 미사용).
- **시트 스키마 v1.1 확장(파도 1부터 적용)**: 유저 관점 디자인 싱킹(F3b) — ①**훅 한 줄**(2인칭 검증형, 원문 화법 재사용) ②**적중검증형**(past_event>trait>timing>none — L3/L4 정렬 규칙화, 파도 1은 이 필드만 우선 실측) ③**파고들기 사다리**(훅→원리→축자+출처 3단). 신뢰 축(F2b) — 몰입층(대사·각주 금지)/신뢰층(접힌 근거창·축자+문헌명) 2층 분리, 미검수 전사 유래 고지는 근거창 안에서만, 정규화 미통과 축자 = 인용 금지·의역만, "이 풀이 이상해요" 피드백 = 계정 축 append, 인용 옆 **내 원국 글자 칩**(조회 키 시각 노출 = 콜드리딩과의 차별).
