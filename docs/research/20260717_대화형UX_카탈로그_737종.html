아키타입대표 앱우리 도사 적용

B. 바로 훔쳐올 Top 5

  1. 피카/Mystic Messenger 패턴 = 도사가 먼저 정곡 던지고 선택지로 응답. 한국 피카(520만 DL)가 "캐릭터 선발화+선택지 진행"으로 이미 검증. 우리 정곡+맞아/아니야와 1:1.
  2. Root Double 에니어그램 공감도 슬라이더 = 유형론을 대화 분기에 실제로 쓴 선례. 우리 "성향(십신·신강약)→딜리버리 라우팅"이 억지가 아님을 증명.
  3. Telltale "X가 기억합니다" + Life is Strange 되감기 = 맞아/아니야에 무게감. "아니야 되돌리기"=명식수 유령가지(리플레이).
  4. Reigns 스와이프+4미터 = 빠른 결정 + 오행 게이지가 대화 중 실시간 출렁.
  5. 호감도 언락(오토메·Grok Ani 애정도 레벨) = 도사 단골이 될수록 깊은 풀이 = 진짜 점집 정석 + 리텐션.

C. 안티패턴 (피할 것 · 수집에서 관찰)

  • 무한 유료 게이트(Episode·Chapters 다이아/에피소드 남발) → 우리는 "무료 밀도 사수, 깊이만 유료".
  • 챗봇 티(AI 자백) → 몰입 파괴. 예타 메타차단 규칙 이식.
  • 겁주기 마케팅(공포 챗스토리·신살 공포) → 흉살=활용/질문형 변환.
  • 콜드리딩(전문가 실시간챗의 막던지기) ↔ 우리는 계산 기반 = 떳떳함.
  • 규제 리스크: 중국 AI 拟인화 규제로 다수 오프라인·터키 Faladdin 대표 구속 = 점술 콘텐츠 톤·고지 주의.

D. 차별화 축 (우리만의 것)

① "진짜 계산" — Spook류는 이름 넣으면 랜덤, 우리는 팔자 결정론. ② 성향 라우팅 — Root Double 선례. ③ 정직 3단 어조 — 계산=단정/해석=hedge/사건=질문 = 콜드리딩과 가르는 선. ④ 무료 도감 — "생일이 뽑는 가챠"로 사행성 회피.
