## 5. 흡수 계획

- **(a) 지식모듈 인코딩**: 011 `성정서사[]` 신설(§3) → 006 삶장면 태그(§3) → 007 근묘화실(0295) → 009 패턴 2종 신규(병존·리스크감당형) → **014 특수신살**(괴강6·백호7 60갑자, 귀문 6쌍, 양인=양일간&제왕) → **015 오행 과다/발달/고립**(0300~0306).
- **(b) 표현층 참고만**: 통념 꺾기 훅("부의 크기는 모른다")·비유 개념(부잣집 머슴, 우발 발동)은 개념만 차용, 문장 재창작. **0003·0295 = "확인 없이 단정 금지"의 교과서 실증**(통설 외향 ↔ 실제 집돌이) — 설계 §5 근거 사례.
- **(c) 배제**: 특정인 풀이(정치인·연예인) · 홍보/멤버십 · 에니어그램 8편(다른 체계 — §6-2 스타일 라우팅 참고로만) · 꿈해몽·자미두수(엔진 범위 밖).
- **(d) 착수 순서**: ①011(커버리지 최대·안전) → ②006+007 → ③009+015 → ④014 → ⑤답변지 26장면을 정곡 후보 풀에 시드.
