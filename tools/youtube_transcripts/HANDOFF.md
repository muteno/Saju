# 🤝 인수인계 — 유튜브 자막 추출 (로컬 Claude Code용)

> **이 문서를 읽는 로컬 Claude Code에게:** 아래 미션을 그대로 이어받아 실행하면 됩니다.
> 클라우드 세션은 유튜브 IP 차단으로 자막 실추출이 불가해서, 실행만 로컬(비차단 IP)로 넘깁니다.
> 도구·리스트·검증까지 전부 준비돼 있으니 **처음부터 새로 짤 필요 없이 그대로 실행**하세요.

---

## 🎯 미션 (한 줄)

`extract_to_desktop.py` 에 **내장된 유튜브 영상 321개(+재생목록 4개)** 의 자막(대화 글자)을 전부 추출해서,
**사용자 바탕화면에 `유튜브_자막_모음` 폴더를 만들고 → 영상별 텍스트 파일 + 전체 묶음 워드 1개**로 저장한다.

## 📤 최종 산출물 (바탕화면/유튜브_자막_모음/)

```
바탕화면/유튜브_자막_모음/
├─ 유튜브_자막_모음.docx     ← 전체를 한 파일로 묶은 워드 (영상마다 제목+주소+자막)
├─ 결과요약.csv              ← 영상별 성공/실패·언어·글자수 표
└─ 텍스트/
   ├─ 0001_영상제목.txt      ← 영상 1개 = 파일 1개 (제목·채널·주소·자막)
   ├─ 0002_영상제목.txt
   └─ … (영상 수만큼)
```

## ▶ 실행 순서 (로컬 Claude Code가 그대로 수행)

1. **위치 이동** — 이 폴더로:
   ```
   cd tools/youtube_transcripts
   ```

2. **필요 패키지 설치** (한 번만):
   ```
   pip install -r requirements.txt
   ```
   (내용: `youtube-transcript-api`, `python-docx`, `yt-dlp`)

3. **실행**:
   ```
   python extract_to_desktop.py
   ```
   - 바탕화면 위치는 자동 탐지(윈도우/맥/OneDrive/한글 '바탕화면' 모두 대응).
   - 특정 폴더로 보내려면: `python extract_to_desktop.py "원하는/폴더/경로"`

4. **진행 확인** — 화면에 `[ 12/325] OK <id> · ko · 1234자 · 제목…` 형태로 한 줄씩 출력됨.

---

## ✅ 검증 (실행 끝나고 로컬 Claude Code가 확인할 것)

1. **개수 대조**: `텍스트/` 안 `.txt` 파일 수가 처리 대상 수와 일치하는지.
   재생목록이 펼쳐지므로 최종 대상 수는 321 + (재생목록 안 영상들)이 됨 — 마지막 요약줄 `완료: 성공 N / 전체 M` 확인.
2. **스팟체크**: `텍스트/0001_*.txt` 몇 개 열어 실제 자막 문장이 들어있는지 눈으로 확인.
3. **워드 열림**: `유튜브_자막_모음.docx` 가 정상 열리고 영상별 섹션이 있는지.
4. **실패 목록**: `결과요약.csv` 에서 '실패'행 확인 → 자막이 꺼진 영상은 어쩔 수 없고,
   **일시 차단으로 실패한 건 스크립트를 한 번 더 실행하면 대개 채워짐**(이미 성공한 파일은 덮어써도 무방).

---

## 🧩 실패가 많을 때 (로컬에서도 간헐 차단 시)

- `extract_to_desktop.py` 상단 `SLEEP_SEC = 1.0` → `2.0`~`3.0` 으로 늘리면 차단이 줄어듦.
- 한국어 자막이 없어 영어로 떨어지는 게 싫으면 `LANG_PREF` 순서 조정.
- 시간표시가 필요하면 `TIMESTAMPS = False` → `True`.
- 재생목록 펼치기가 필요 없으면 `EXPAND_PLAYLISTS = False`.

## ℹ️ 배경 (왜 로컬로 넘기나)

클라우드(원격) 세션에서 자막 자동추출을 시도했으나 **유튜브가 데이터센터 IP를 봇으로 차단**함:
`youtube-transcript-api` → RequestBlocked · `yt-dlp` → "Sign in to confirm you're not a bot" ·
watch 직접요청 → 302 → `google.com/sorry`(구글 비정상트래픽 차단). **집/회사 일반 IP에서는 정상.**
그래서 도구·리스트·검증을 전부 준비해두고 **실행만** 로컬로 인계하는 것.
