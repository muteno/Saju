# 유튜브 자막(글자) 추출 → 영상별 저장 + 워드 파일

보내준 유튜브 주소 **321개(+재생목록 4개)**의 자막(영상 속 말을 글자로 옮긴 것)을 통째로 뽑아서,
**영상 1개당 텍스트 파일 1개**로 저장하고, 마지막에 **전부 묶은 워드 파일 1개**를 만들어 주는 도구다.

> ## ⭐ 바탕화면에 바로 저장하고 싶으면 → `extract_to_desktop.py`
> 리스트가 **스크립트 안에 내장**돼 있어 파일 하나만 있으면 되고, 결과를 **바탕화면 `유튜브_자막_모음` 폴더**에 저장한다.
> ```
> cd tools/youtube_transcripts
> pip install -r requirements.txt
> python extract_to_desktop.py
> ```
> **로컬 Claude Code에 넘길 때는 `HANDOFF.md`** 를 읽히면 그대로 이어받아 실행한다.
> (아래 `extract.py` 는 결과를 레포 `output/` 폴더에 저장하는 버전 — 취향대로 선택.)

---

## ⚠ 먼저 꼭 알아둘 것 — 왜 "내 컴퓨터"에서 돌려야 하나

이 작업은 클라우드(Claude가 돌아가는 원격 서버)에서 **자동 실행이 막혔다.**
유튜브가 데이터센터 IP에서 오는 자동 자막 요청을 "봇"으로 보고 차단하기 때문이다.
(확인됨: 요청이 구글의 `google.com/sorry` 차단 페이지로 튕겨나감.)

**집이나 회사의 일반 인터넷에서는 대부분 정상 동작한다.** 그래서 이 폴더를 통째로 받아
네 컴퓨터에서 아래 순서대로 실행하면 된다.

---

## 준비물 (한 번만)

컴퓨터에 **파이썬(Python)** 이 필요하다 — 파이썬 = 이 스크립트를 실행해 주는 프로그램.
- 없으면: <https://www.python.org/downloads/> 에서 내려받아 설치.
  (윈도우 설치 시 **"Add Python to PATH"** 체크박스 꼭 켜기 — 이게 "어디서든 파이썬을 부를 수 있게" 하는 설정.)

---

## 실행 방법 (순서대로 따라 하기)

### 1) 이 폴더로 이동
- **터미널**(윈도우는 "명령 프롬프트" 또는 "PowerShell", 맥은 "터미널") = 컴퓨터에 글자로 명령을 내리는 검은 창.
- 그 창에 아래를 치고 Enter (경로는 이 폴더 위치에 맞게):

```
cd 경로/tools/youtube_transcripts
```

### 2) 필요한 부품(패키지) 설치 — 한 번만
```
pip install -r requirements.txt
```
(`pip` = 파이썬 부품을 자동으로 깔아주는 도구. 위 한 줄이면 자막 추출·워드 생성에 필요한 3개가 다 깔린다.)

### 3) 실행
```
python extract.py
```
- 그러면 화면에 `[ 1/325] OK ...` 처럼 한 줄씩 진행 상황이 뜬다.
- 영상 수가 많아 **시간이 좀 걸린다**(영상 사이에 일부러 1초씩 쉼 — 차단 안 당하려고).

### 4) 결과 확인
실행이 끝나면 `output` 폴더에 이렇게 생긴다:
```
output/
├─ 유튜브_자막_모음.docx     ← ★ 전부 묶은 워드 파일 (열어서 바로 확인)
├─ 결과요약.csv              ← 어떤 영상이 성공/실패했는지 표
└─ 텍스트/
   ├─ 0001_영상제목.txt      ← 영상별로 하나씩 저장된 글자
   ├─ 0002_영상제목.txt
   └─ ...
```

## 자주 나오는 질문

- **"자막을 가져오지 못함" 이라고 뜬 영상이 있어요.**
  → 그 영상은 자막 자체가 꺼져 있거나(업로더가 안 켬), 일시적 차단이다. 다시 실행하면 잡히는 경우도 있다.
  성공/실패는 `결과요약.csv` 에서 한눈에 볼 수 있다.

- **한국어 말고 영어 자막이 나와요.**
  → 한국어 자막이 없으면 자동으로 영어를 가져온다. `extract.py` 위쪽 `LANG_PREF` 순서를 바꾸면 우선순위 조절 가능.

- **시간까지 같이 넣고 싶어요 (예: [01:23] ...).**
  → `extract.py` 위쪽 `TIMESTAMPS = False` 를 `True` 로 바꾸면 각 줄 앞에 시간이 붙는다.

- **주소를 더 넣거나 빼고 싶어요.**
  → `youtube_urls.txt` 파일을 열어 한 줄에 유튜브 주소 하나씩. `#` 으로 시작하면 주석(무시).

## 다른 방법 (내 컴퓨터가 어려우면)

브라우저 쿠키(로그인 정보)를 내보내 주면, 클라우드 세션에서도 우회 실행이 가능하다.
다만 쿠키는 민감 정보라 다루기 번거롭다 — **위의 "내 컴퓨터에서 실행"이 가장 쉽고 안전하다.**
필요하면 알려줘, 쿠키 방식도 안내할게.
